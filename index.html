训练配置（两次运行相同）

模型 / 架构

运动编码器 chunk-based，motion_mode=local
motion dim 384，M=1 token；hidden 768 / depth 8 / heads 12
spatial_pool=false，encoder_stride=2
DiT 解码器 hidden 768 / depth 4 / heads 12，window_size=7，local
双流 SigLIP 1152 ⊕ DINOv3 1280 → encoder 输入 2432
状态分支 StateResidualMLP，13 维 delta

chunk / 优化 / 正则

chunk K = max_horizon = 16，stride=1，delta_mode=from_current
优化 AdamW lr 3e-4，wd 1e-4，fp32，batch 8/卡，max_epochs 40
正则 SIGReg 把 motion 推向 N(0,I)，带 warmup
运动过滤 token_mag ≥ 0.97（StopCube override 0.8，仅含 StopCube 的 v2 生效）
EMA decay 0.999，验证 / 可视化用 EMA 权重
损失 siglip·MSE + dino·MSE + 0.003·SIGReg + 1.0·MSE(state)

数据集格式（两版本结构一致，按 episode 一组 raw .bin）

tokens/ SigLIP，bf16，每帧 256 patch × 1152 dim
dino-tokens/ DINOv3（vith16plus），bf16，每帧 256 × 1280 dim
states/ fp32，13 维归一化本体感知 delta（bf16 尾数太粗，故用 fp32）
norm_stats.json 状态分位归一 q01/q99 → [-1,1]；token_delta_rms / dino_token_delta_rms 两路各自除 RMS 后 concat
chunk_motion.npz 每 chunk 运动幅度（运动过滤用）
metadata.json 每 episode 的 frame_stride 与 source_video
抽取参数 image 256 / patch 16 / frame_stride 2；chunk 级 seek 读取，零内存驻留

读取按 __getitem__ seek 取 K+1 帧并预计算 delta；token-delta 与 state-delta 经线性投影加法融合进编码器。
