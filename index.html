共享骨干（所有 run 相同）

模型 / 架构

运动编码器 chunk-based，motion_mode=local
motion dim 384，M=1 token；hidden 768 / depth 8 / heads 12
spatial_pool=false，encoder_stride=2
DiT 解码器 hidden 768 / depth 4 / heads 12，window_size=7，local
多流 各路 delta 各自除 RMS 后 concat 喂共享 encoder；每路一个镜像解码器
状态分支 StateResidualMLP，13 维 delta

chunk / 优化 / 正则

chunk K = max_horizon = 16，stride=1，delta_mode=from_current
优化 AdamW lr 3e-4，wd 1e-4，fp32，max_epochs 40
正则 SIGReg 把 motion 推向 N(0,I)，带 warmup
运动过滤 token_mag ≥ 0.97（StopCube override 0.8）
EMA decay 0.999，验证 / 可视化用 EMA 权重
光流（三流） encoder 输入 (K,256,2)，按 flow_rms 归一、不做 delta；sliding=anchor 单条 record，sum=neighbour 相邻帧对累积（flowSumDense 再用 ConvexUpSample 重建 512 维稠密光流）

数据集格式（按 episode 一组 raw .bin，chunk 级 seek 读取）

tokens/ SigLIP，bf16，每帧 256 patch × 1152 dim
dino-tokens/ DINOv3（vith16plus），bf16，每帧 256 × 1280 dim
flow-tokens-*/ 光流（仅 v4），fp32，anchor-indexed 每条 record (K,256,2)；sliding / sum 两种
states/ fp32，13 维归一化本体感知 delta
norm_stats.json 状态分位归一 q01/q99 → [-1,1]；token_delta_rms / dino_token_delta_rms / flow_rms_{sliding,sum} 各自归一
chunk_motion.npz 每 chunk 运动幅度（运动过滤用）
metadata.json 每 episode 的 frame_stride 与 source_video
抽取参数 image 256 / patch 16 / frame_stride 2；零内存驻留

读取按 __getitem__ seek 取 K+1 帧并预计算 delta（光流为按 start 取单条 anchor record）； 多路特征 concat 后喂共享 encoder 压成 motion token。
